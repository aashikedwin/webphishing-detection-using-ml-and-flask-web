About This Project

The Phishing Detection System is an intelligent web application designed to detect and classify suspicious URLs as "Phishing" or "Legitimate" using advanced Machine Learning models. With the growing number of phishing attacks targeting individuals and organizations, this system aims to provide a quick and reliable way to analyze URLs and protect users from falling victim to malicious sites.

Project Objectives:

  • To develop an effective URL phishing detection tool using Machine Learning techniques.
  • To analyze and classify URLs in real-time with high accuracy and low false positives.
  • To leverage advanced algorithms like Gradient Boost, Random Forest, SVM, and Neural Networks to enhance detection performance.
  • To design a user-friendly web interface for users to check URLs effortlessly.

Key Features:

  • Fast URL analysis and classification as "Phishing" or "Legitimate".
  • Backend powered by Python Flask and integrated with MongoDB for result storage.
  • Machine Learning models trained on merged datasets from PhishTank and University of New Brunswick Dataset.
  • 30 critical features extracted from URLs (Address bar, Domain-based, and HTML/JavaScript-based).
  • Interactive dashboard to review recent URL classifications and history.

Technologies Used:

  • Python for backend processing and Machine Learning model implementation.
  • Flask as a lightweight web framework to serve the application.
  • MongoDB as a NoSQL database to store URL analysis records and user feedback.
  • Machine Learning Libraries: Scikit-learn, XGBoost, LightGBM, TensorFlow/Keras for Autoencoders.
  • Frontend: HTML, CSS, and JavaScript for responsive web design and AJAX-based URL checking.

Project Highlights:

  • Compares 7 different Machine Learning models including ensemble and deep learning models for performance.
  • Automatically identifies legitimate government, educational, and organizational domains.
  • Clean and responsive user interface suitable for desktops and mobile devices.
